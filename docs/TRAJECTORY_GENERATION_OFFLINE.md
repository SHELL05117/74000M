# 路径与轨迹生成（离线）

状态：固定容量几何与时间参数化 `PCValidated`；任何数值均不代表真机可用极限。

## 实现边界

- 路径点包含世界坐标 `Pose2d`、切线尺度和前进/后退方向；三次 Hermite 插值从一阶/二阶导数计算航向与曲率。
- 同一条轨迹保持一个行驶方向；方向切换必须在零速处分成两个命令/轨迹，生成器对混合方向明确返回 `DirectionChangeUnsupported`。
- 前向/后向时间参数化同时限制线速度、加速、减速、角速度、向心加速度、总抓地加速度、差速轮速度比例和 `kS/kV/kA` 预测电压。
- 支持非零初速与指定终速；若制动距离、曲率、容量或电压预算不可行，返回失败，不静默改写端点。
- 路径、工作数组和轨迹全部为编译期固定容量；容量不足明确失败。控制环只调用 `sampleAt()`，不生成路径、不分配内存。
- 任意合法时间周期可重采样；终点显式追加，10 ms/20 ms 等采样不会丢失终点。

PC 测试覆盖直线、圆弧近似、S 路径、正反、非零初速、端点、连续性、约束、采样周期一致性、容量和非法方向。真机必须从实测轮速、抓地、左右前馈、电压余量和机构载荷确定约束；当前 trajectory 只能用于仿真/Fake/replay，不能开启 `autonomous_motion`。
